# Contract Agent (Reference)

## Papel
Analisar e validar o contrato OpenAPI
entre back-end e front-end.

---

## Escopo
- Revisar `contracts/openapi.v1.yaml`.
- Identificar inconsistências de payload.
- Sugerir ajustes de contrato.

---

## Observação
Este agent **não implementa código**
e **não atua automaticamente** no front-end.
